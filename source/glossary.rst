
Глоссарий
=========

.. glossary::
   :sorted:

   login
      Имя Партнера в системе. Сервис-провайдер создает имя Партнера и предоставляет его Партнеру при подключении сервиса.

   password
      Пароль Партнера в системе. Сервис-провайдер создает пароль и предоставляет его Партнеру при подключении сервиса.

   MO-сообщение 
      (Mobile Originated) - сообщение, которое отправляет абонент со своего устройства в сторону сервера. 

   MT-сообщение 
      (Mobile Terminated) - сообщение, отправляемое со стороны сервера на устройство абонента (текст до 1000 символов, изображение, гипертекстовая ссылка в виде кнопки).

   FlashingCall
      Звонок на телефон абонента со специальной номерной ёмкости. Последние цифры номера вызывающего абонента рассматриваются в качестве кода.

   VoiceCode	
      Голосовое воспроизведение входящего сообщения, содержащего код авторизации.

   URI 
      (Uniform Resource Identifier) - универсальный идентификатор ресурса. Сервис-провайдер предоставляет Партнеру.

   URL 
      (Uniform Resource Locator) - универсальный указатель ресурса. Партнер предоставляет Сервис-провайдеру.

   Партнер
      Организация, подключенная к платформе Сервис-провайдера для передачи/приема сообщений, а также приема статусов сообщений.

   Сервис-провайдер
      Организация, предоставляющая Партнерам сервисы по приему/передаче сообщений и их статусов посредством своей технологической платформы.

   Каскадная рассылка
      Последовательная отправка сообщения через разные каналы в течение времени жизни сообщения. Переотправка сообщения производится в случае получения статуса неуспешно отправленного сообщения.

   Время жизни сообщения
      Интервал, в течение которого оператор будет пытаться доставить сообщение до абонента. Для каждого типа сообщения настраивается отдельно.   
  
   serviceId
      Уникальный идентификатор сервиса при интеграции посредством HTTP API. Сервис-провайдер предоставляет Партнеру при подключении сервиса.

   pass
      Пароль для авторизации в сервисе при интеграции посредством HTTP API. Сервис-провайдер предоставляет Партнеру при подключении сервиса.

   Длинная ссылка
      Ссылка, размер которой превышает количество знаков, указанных в настройке сервиса. По умолчанию установлено значение: 26 символов.

   Короткая ссылка
      Ссылка, полученная путем сокращения длинных ссылок.

   Домен
      Доменное имя, предоставляемое Партнером, которое будет подставляться в короткую ссылку.

   UID
      Уникальный идентификатор длиной от 3 до 8 символов, который добавляется после домена для формирования уникальной ссылки.
