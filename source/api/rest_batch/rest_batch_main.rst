REST_BATCH API
===============

REST_BATCH API v.3.0 предназначен для интеграции в систему Партнёра.

При интеграции по REST_BATCH API Партнёр получает следующие возможности:

* отправку как одиночных SMS-сообщений, так и пакета SMS-сообщений;
* передачу статусов сообщений по нотификационной схеме;
* получение статусов по идентификаторам SMS-сообщений или по идентификатору пакета SMS-сообщений.
