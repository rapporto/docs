
IVR
=======
 
.. toctree::
   :maxdepth: 1
   :hidden:

В сервисе :term:`IVR` взаимодействие проиcходит по определенному сценарию, управлять которым абонент может посредством нажатий на определенные клавиши на телефоне, т.к. каждой клавише соответствует определенный код :term:`DTMF`. 

Абоненту воспроизводится голосовое сообщение, при этом обработка нажатий клавиш не производится.

В :abbr:`RCS (Rapporto Communication System)`-платформе предусмотрена передача :abbr:`IVR (Interactive Voice Response)`-сообщений в направлении от Партнера к абоненту по двум протоколам:
 
* :doc:`ivr_ftp/ivr_ftp_main` позволяет Партнеру создавать и запускать на исполнение рассылки через загрузку файла установленного формата на Платформу RCS. Загрузка файла производится любым методом, обеспечивающим достаточную безопасность, например, по :abbr:`FTPS (также FTP/SSL — File Transfer Protocol + SSL (Secure Sockets Layer))` или :abbr:`FTP (File Transfer Protocol)` через ipsec-туннель.
* :doc:`ivr_http/ivr_http_main` прост в реализации и удобен в отладке.
