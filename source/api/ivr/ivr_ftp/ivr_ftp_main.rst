:abbr:`IVR (Interactive Voice Response)` :abbr:`FTP (File Transfer Protocol)` :abbr:`API (Application Programming Interface)` v.5.0 предназначен для использования в качестве интерфейса обмена файлами для создания IVR-рассылок на платформе :abbr:`RCS (Rapporto Communication System)`.

Функциональные возможности
----------------------------

IVR FTP API обеспечивает возможность авторизованному Партнеру:

* автоматизированным образом создавать и, при наличии соответствующих привилегий, запускать на исполнение рассылки через загрузку файла установленного формата на Платформу;
* производить безопасную загрузку файлов по FTPS или FTP через ipsec-туннель;
* загружать рассылки с идентификаторами;
* включить генерацию файлов с отчетами о финальных статусах звонков.
